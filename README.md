## domain model
Everyone has an ID
A User (class)
- username String (attribute)
- password String (attribute)

A Post (class)
- title String
- text String
- madeBy String
- publishedAt Time
- comments List of Comments

A comment (class)
- commenter String
- publishedAt Time
- stars Int
- text String
